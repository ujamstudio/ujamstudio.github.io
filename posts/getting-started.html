Setting Up My Blog with GitHub Pages

Projects

Why GitHub Pages?

I wanted a simple, free, and fast way to publish my study notes and project logs. GitHub Pages checks all the boxes — it serves static files directly from a repository with zero configuration.

Project Structure

The blog is organized like this:

ujamstudio_blog/
├── index.html          # Home page with post list
├── about.html          # About me
├── style.css           # Global styles
└── posts/
    └── getting-started.html
